Golang Package

以 GitHub 为例，发布一个 module 需要创建一个以 github.com/username/mymodule 为名称的mod文件。

运行命令以创建 go.mod 文件：

然后就可以开始编写模块。

模块编写完成之后运行 go mod tidy 和 go test ./... 命令确保一切正常。

然后使用 git tag 命令新建版本号，并推送到远程仓库：
